# 诗境通 · 功能规格文档 v1.0
> AI赋能学生学习中国古诗词 · Web应用 · 竞赛版本

---

## 一、项目定位与目标用户

### 1.1 产品定位
**诗境通**是一款面向中小学生的 AI 古诗词智慧学习平台，通过大语言模型赋能，将"学、记、练、测、创"五个环节整合为完整的学习闭环，解决学生在古诗词学习中的核心痛点。

### 1.2 目标用户
| 用户类型 | 核心诉求 | 优先级 |
|---------|---------|-------|
| 初中生（核心） | 中考备考、理解赏析、背诵效率 | P0 |
| 高中生 | 高考古诗词板块、深度鉴赏 | P0 |
| 小学生 | 趣味启蒙、基础背诵 | P1 |
| 语文教师 | 课堂辅助、出题参考 | P2 |

### 1.3 核心价值主张
- **快**：输入任意诗词，秒级获得完整解析
- **准**：AI精准定位学生薄弱点，针对性强化
- **活**：对话式学习，替代枯燥背诵，激发兴趣
- **全**：从理解到创作，覆盖学习全链路

## 三、各模块功能详细规格

---

### M1：AI 智慧解析

#### 功能描述
用户输入任意古诗词（全文或部分），AI 一键生成多维度解析内容，是整个平台的核心入口。

#### 子功能清单

**M1-01 诗词检索与录入**
- 搜索框支持：诗名、作者名、诗句关键词模糊搜索
- 系统内置数据库：3000+ 常用古诗词（覆盖小中高教材）
- 支持用户手动粘贴任意诗词全文
- 快速入口：教材同步（按年级/出版社分类浏览）

**M1-02 一键深度解析**

AI 解析输出结构（固定格式，前端渲染）：

```
① 基本信息卡
   - 题目、作者、朝代、体裁（五绝/七律/词/曲等）
   - 创作时间（如知）、创作背景简述

② 注释与译文（逐句对照）
   - 重点字词注释（可展开/收起）
   - 全诗现代文译文
   - 逐联/逐句精准对照

③ 意象与意境分析
   - 列举核心意象（如：明月→思乡、菊花→隐逸）
   - 整体意境概括（50字以内）

④ 艺术手法分析
   - 修辞手法：比喻、拟人、对偶、用典等（举句说明）
   - 表现手法：虚实结合、借景抒情、托物言志等

⑤ 情感主旨
   - 作者情感（忧国伤时/思乡离别/志向抱负等）
   - 主题一句话概括

⑥ 作者与背景
   - 作者小传（100字）
   - 创作背景（所处朝代、社会背景、个人经历）

⑦ 考点提示（中考/高考向）
   - 高频考查角度（3-5条）
   - 易错点提醒
   - 相似诗词推荐（可跳转）
```

**M1-03 解析深度控制**
- 「简版」：仅译文 + 情感主旨（适合小学生）
- 「标准版」：全部8项内容（默认）
- 「考试版」：强化考点，突出答题模板

**M1-04 收藏与笔记**
- 用户可收藏诗词到「我的诗集」
- 支持在解析结果上添加个人笔记（富文本）
- 笔记自动保存，下次打开可见

#### 交互逻辑
```
用户进入解析页
→ 搜索/粘贴诗词
→ 选择解析深度
→ 点击「开始解析」
→ AI流式输出（逐块显示，不等待全部完成）
→ 解析完成后：出现「练一练」「聊一聊」「背一背」快捷按钮
→ 点击跳转对应功能模块
```

#### AI Prompt 设计要点
- 系统prompt指定输出为结构化JSON，前端按字段渲染
- 支持流式输出（SSE），逐块呈现提升体验感
- 内置角色：「你是一位专业的语文老师和古典文学研究者，擅长用通俗易懂的语言解析古诗词」

### M2：对话式 AI 导师

#### 功能描述
学生可以与 AI 进行自由对话，深化对诗词的理解。提供两种对话模式：常规「问答模式」和特色「诗人穿越模式」。

#### 子功能清单

**M2-01 自由问答模式**
- 学生可就任意诗词提问（「这首诗为什么用"举头望明月"而不是"低头"？」）
- AI 采用苏格拉底式引导：不直接给答案，先反问引导思考
  - 例：「你觉得诗人当时在哪里？那个场景给你什么感觉？」
- 支持追问、反驳、延伸话题
- 对话历史自动保存（按诗词分类归档）

**M2-02 诗人穿越模式（特色功能）**
- AI扮演历史诗人（李白、杜甫、苏轼、李清照、王维等30+诗人）
- 学生以第一人称与诗人对话
- AI基于：诗人生平、性格特征、所处时代、代表作品，生成符合人物性格的回应
- 示例对话：
  - 学生：「李白，你为什么要写《静夜思》？」
  - AI-李白：「那是天宝十五年的秋夜，我旅居扬州，独坐窗前，月光铺满地板，恍若白霜——你可曾在异乡的夜里，突然想起家乡的某个画面？」
- 角色切换时，AI自动调整：语气/词汇/历史背景参照系

**M2-03 现代共鸣引导**
- AI 将古诗情感与学生现代生活关联
- 例：「《送元二使安西》写的是送别，你有没有经历过类似的离别？那种心情和王维有什么相似？」
- 通过情感共鸣建立记忆锚点，提升长期记忆效果

**M2-04 对话学习小结**
- 每次对话结束（或主动触发），AI自动生成：
  - 「本次对话中你理解了：XXX」
  - 「你提出的好问题：XXX」
  - 「还可以继续探索：XXX」
- 小结可导出到笔记

#### 交互逻辑
```
进入「聊一聊」（来自M1的快捷跳转，或独立入口）
→ 选择对话模式：问答模式 / 诗人穿越
→（穿越模式）选择诗人角色
→ 当前诗词自动加载到上下文
→ 开始对话（支持语音输入，可选）
→ AI回复（流式，打字机效果）
→ 对话结束 → 「查看小结」→「去练习」
```

#### AI Prompt 设计要点
- 诗人模式：system prompt 注入诗人画像（生平/性格/时代/代表作），要求AI保持人物一致性
- 引导模式：注入苏格拉底式引导指令，禁止直接给出完整答案
- 上下文窗口：携带当前诗词全文 + 解析内容 + 对话历史

### M3：智能记忆系统

#### 功能描述
基于遗忘曲线和间隔重复原理，帮助学生高效背诵古诗词，减少"背了就忘"的问题。

#### 子功能清单

**M3-01 记忆模式选择**
- **逐句填空**：显示诗句，遮盖部分词，学生填写
- **上句接下句**：显示上句，学生输入下句
- **全文默写**：空白页面，学生完整默写（支持手写识别，可选）
- **听写模式**：AI朗读，学生默写

**M3-02 智能提示系统**
- 首字提示：显示每句第一个字
- 韵脚提示：显示每句最后一字
- 节奏提示：显示格律断句
- 遇忘触发：3秒无输入 → 自动显示下一级提示

**M3-03 间隔重复调度**
- 记录每首诗每次记忆的时间和正确率
- 基于 SM-2 算法计算下次复习时间
- 「今日待复习」列表：每日推送需复习的诗词
- 记忆强度可视化：每首诗显示记忆曲线

**M3-04 记忆成就系统**
- 连续记忆天数（打卡）
- 已掌握诗词数量
- 本周记忆量统计

#### 交互逻辑
```
进入「背一背」
→ 显示待记忆列表（新诗词 + 今日复习）
→ 选择记忆模式
→ 开始记忆练习
→ 完成后：AI评分 + 下次复习时间提示
→ 返回列表，更新记忆状态
```

### M4a：AI 智能题目生成

#### 功能描述
针对任意诗词，AI 自动生成覆盖多维考查角度的练习题，替代人工出题，实现即学即练。

#### 题目类型

| 题型 | 描述 | 考查维度 | 适用学段 |
|-----|------|---------|---------|
| 默写填空 | 给出部分诗句，填写空白处 | 记忆 | 全学段 |
| 词义辨析 | 选择字词正确解释 | 理解 | 小学-初中 |
| 手法判断 | 判断修辞/表现手法是否正确 | 鉴赏 | 初中-高中 |
| 情感分析 | 选择/简答情感主旨 | 赏析 | 初中-高中 |
| 名句赏析 | 对指定名句进行赏析分析 | 表达 | 高中 |
| 比较阅读 | 与另一首诗比较异同 | 综合 | 高中 |
| 语境默写 | 给出情景，写出对应名句 | 活用 | 初中-高中 |

**M4a-01 智能生成规则**
- 选择诗词后，AI根据诗词特点自动决定生成哪些题型
- 可手动设置：题型偏好、难度（简单/中等/困难）、题量（5/10/15题）
- 生成格式：JSON结构化输出（题干、选项、正确答案、解析）

**M4a-02 题目质量保障**
- AI生成后进行自我校验（再次调用AI验证答案正确性）
- 题目难度标注（⭐一颗星到⭐⭐⭐⭐⭐五颗星）
- 用户可标记「题目有误」，反馈改进

**M4a-03 题目集管理**
- 生成的题目集自动保存
- 可再次练习、分享（生成分享链接）
- 教师角色可批量导出题目（PDF格式）

#### 交互逻辑
```
在诗词详情页 → 点击「练一练」
→ 弹出题目生成设置（题型/难度/题量）
→ AI生成题目（5-10秒，显示加载动画）
→ 进入答题界面（逐题显示）
→ 每题答完立即显示：正确/错误 + AI详解
→ 全部完成 → 成绩统计 + 弱点标注 → 录入错题本
```

### M4b：弱点精准诊断

#### 功能描述
系统持续追踪学生的答题数据，自动识别薄弱点，生成可视化诊断报告，指导下一步学习方向。

#### 诊断维度

```
弱点诊断维度树
├── 按题型维度
│   ├── 默写准确率（记忆弱点）
│   ├── 词义理解准确率（理解弱点）
│   ├── 手法辨析准确率（鉴赏弱点）
│   └── 赏析表达得分（表达弱点）
├── 按朝代维度
│   ├── 唐诗掌握度
│   ├── 宋词掌握度
│   ├── 其他朝代掌握度
├── 按题材维度
│   ├── 边塞诗、田园诗、送别诗...
│   └── 各类题材正确率
└── 按意象维度
    ├── 常见意象掌握情况
    └── 高频错误意象统计
```

**M4b-01 实时弱点追踪**
- 每次答题后，实时更新弱点数据
- 后台维护：每个诊断维度的历史得分记录

**M4b-02 弱点诊断报告**
- 雷达图：展示各维度得分（前端用 Recharts 渲染）
- 弱点排行：「你最薄弱的3个方向：XXX」
- 趋势图：每周进步/退步情况
- AI 分析语：「你在宋词的情感理解方面较弱，建议重点练习苏轼、李清照的作品」

**M4b-03 智能推荐**
- 根据弱点自动推荐：
  - 相关诗词（「针对你的边塞诗弱点，推荐练习这5首」）
  - 相关题目（直接进入练习）
  - 知识点梳理（调用M1解析）

#### 交互逻辑
```
学生完成若干练习后 → 系统自动更新弱点数据
主动入口：「我的学习」→「弱点诊断」
→ 查看雷达图 + 弱点列表
→ 点击某个弱点 → 查看详情 + 推荐练习
→ 点击推荐诗词 → 跳转M1解析 + M4a练习
```

### M4c：模拟考试系统

#### 功能描述
按照中考/高考真题格式，AI 自动生成完整古诗词模拟卷，提供计时、阅卷、详解的完整考试体验。

#### 考卷规格

**中考模式**
- 题量：古诗词部分约 8-12 题
- 题型：默写填空（必考）+ 诗词鉴赏（2道）
- 时间：20分钟
- 满分：20分（参照各地中考比例）

**高考模式**
- 题量：名句默写3句 + 古诗鉴赏2道大题
- 题型：情感分析 + 手法赏析 + 语境默写
- 时间：30分钟
- 满分：20分（参照高考占比）

**自定义模式**
- 用户自定义：题型、题量、难度、时间、诗词范围

**M4c-01 考试流程**
1. 进入考试中心 → 选择考试类型
2. AI 生成考卷（考前不透露答案）
3. 计时开始（页面顶部显示倒计时）
4. 学生作答（支持文本输入，主观题支持多行）
5. 提交 → AI 阅卷（客观题秒判，主观题 AI 评分+人工核对说明）
6. 查看成绩单 + 详细解析
7. 一键将错题导入错题本

**M4c-02 AI 阅卷规则**
- 客观题：严格匹配，容错异体字（如"飞"/"飞"）
- 主观题（赏析题）：
  - AI 对照标准答案给出得分点
  - 给出 0-5 分的评分（按得分点覆盖数）
  - 给出「本题参考答案」和「得分要点解析」
- 整体反馈：「本次模考你得了XXX分，相当于XX水平，弱项在XXX」

**M4c-03 考试记录**
- 历次模考成绩时间线
- 进步趋势图（折线图）
- 各题型得分率统计

### M5：错题本与复习计划

#### 功能描述
自动收录所有答错的题目，AI 分析错误模式，制定个性化复习计划，实现长效提升。

#### 子功能清单

**M5-01 智能错题本**
- **自动收录**：M4a 练习、M4c 模考中答错的题目自动进入错题本
- **手动添加**：学生可手动将任意题目添加到错题本
- 错题展示：题目 + 我的答案（红色标注）+ 正确答案 + AI详解
- 错误归因标签：AI自动标注错误原因（记忆失误/理解偏差/手法混淆/粗心）
- 错题状态：待复习 / 已掌握 / 需再练
- 支持按：诗词/题型/错误原因/时间 筛选

**M5-02 错题复习模式**
- 「复习我的错题」：从错题本中随机抽取待复习题
- 「专项复习」：按诗词/题型筛选后练习
- 复习后标记掌握状态，已掌握的题目自动降低复习频率

**M5-03 AI 智能复习计划**
- 输入：考试时间（如「距中考还有30天」）
- 系统生成：
  - 每日复习任务（每日10-15分钟量）
  - 优先级排序（弱点优先、高频考点优先）
  - 本周重点诗词列表
  - 阶段性目标（第1周/第2周/考前3天）
- 计划可手动调整（拖拽排序）
- 每日打卡进度追踪

**M5-04 复习提醒**
- 每日任务角标提示（未完成任务数）
- 可选：浏览器通知提醒（用户授权后）

#### 交互逻辑
```
错题本入口：「我的学习」→「错题本」
查看错题列表 → 筛选 → 选择复习模式 → 完成复习 → 更新状态

复习计划入口：「我的学习」→「复习计划」
→ 输入考试信息（或已保存）
→ 查看今日任务 → 开始今日学习 → 完成打卡
```

### M6：AI 创作辅导

#### 功能描述
引导学生从「鉴赏者」成为「创作者」，AI辅助进行古诗词仿写，培育高阶语文素养。

#### 子功能清单

**M6-01 仿写引导**
- 学生选择目标风格（豪放/婉约/田园/边塞/咏物）
- 参考诗词：选择一首经典作为仿写对象
- AI 提供：
  - 结构模板（格律框架、起承转合）
  - 意象建议（该风格常用意象池）
  - 开头引导（提供2-3个开头选项）
  - 字数/韵律提示
- 学生在编辑框中逐句创作
- 实时提示：AI检测格律是否符合（可关闭）

**M6-02 AI 专业点评**
- 提交后，AI从以下维度点评（结构化输出）：
  - **意境**：画面感和情感是否传达（评分1-10）
  - **格律**：是否符合所选诗体规范（评分1-10）
  - **炼字**：有无精彩用词，有无明显病字（评分1-10）
  - **整体评价**：50字综评
  - **改进建议**：2-3条具体修改意见
  - **亮点肯定**：至少指出1处做得好的地方

**M6-03 现代转古体**
- 学生用现代语言描述一个场景/心情
- AI将其「诗化」，提供古体诗版本
- 学生学习AI的诗化逻辑，尝试自己再写一遍

**M6-04 作品展示**
- 个人作品集（可设公开/私密）
- 可选：公开作品到「创作广场」（展示用，竞赛演示亮点）

---

### M7：诗词知识图谱

#### 功能描述
可视化展示诗词的知识关联网络，帮助学生建立系统性认知，而非孤立记忆。

#### 子功能清单

**M7-01 诗人关系图**
- 节点：诗人（大小代表知名度/作品数）
- 边：师承关系、同代关系、唱和关系
- 点击诗人节点：弹出简介 + 代表作列表

**M7-02 意象关联图**
- 核心意象节点（明月/菊花/杨柳/鸿雁等）
- 关联诗词（点击意象看哪些诗用了它）
- 意象情感标注（明月→思乡/感时）

**M7-03 时间轴视图**
- 横轴：朝代时间线
- 纵轴：诗词数量/重要诗人
- 可筛选：按体裁/题材/学段

**M7-04 个人知识图**
- 展示：我已学过的诗词节点
- 连线：相同诗人/意象/风格的诗词
- 帮助学生看到自己知识体系的「完整度」

## 五、数据模型设计

### 5.1 核心数据实体

**Poem（诗词）**
```
id, title, author, dynasty, content（原文）, type（体裁）,
tags（题材标签[]）, grade_level（适合学段[]）,
source（教材来源）, difficulty（难度1-5）
```

**Analysis（解析内容）**
```
id, poem_id, annotation（注释JSON）, translation（译文）,
imagery（意象[]）, techniques（手法[]）, emotion（情感）,
background（背景）, exam_points（考点[]）, created_at
```

**User（用户）**
```
id, name, grade（年级）, target_exam（目标考试）,
streak_days（连续学习天数）, total_poems_learned, created_at
```

**UserPoemRecord（学习记录）**
```
id, user_id, poem_id, memorized_level（0-5）,
last_reviewed_at, next_review_at（间隔复习时间）,
notes（笔记）
```

**Question（题目）**
```
id, poem_id, type（题型）, content（题干）,
options（选项JSON，客观题）, answer（答案）,
explanation（解析）, difficulty（1-5）, source（AI生成/人工）
```

**UserAnswer（答题记录）**
```
id, user_id, question_id, user_answer, is_correct,
time_spent（答题时长秒）, created_at
```

**WrongQuestion（错题本）**
```
id, user_id, question_id, wrong_answer,
error_type（错误类型标签）, status（待复习/已掌握）,
review_count（复习次数）, last_reviewed_at
```

**ExamRecord（考试记录）**
```
id, user_id, exam_type（中考/高考/自定义）,
total_score, max_score, answer_detail（JSON）,
created_at
```

**ReviewPlan（复习计划）**
```
id, user_id, exam_date, daily_tasks（JSON）,
completion_rate, created_at
```

**Creation（创作作品）**
```
id, user_id, title, content（作品内容）,
style（风格）, reference_poem_id, ai_feedback（JSON）,
is_public, created_at
```

## 八、非功能要求

### 8.1 性能
- AI解析首字符响应时间：< 2秒（流式输出）
- 题目生成时间：< 10秒
- 页面初始加载：< 3秒

### 8.2 稳定性
- AI调用失败：自动重试2次，超时后显示友好提示
- 所有AI生成内容缓存：相同诗词的解析结果缓存（Supabase），避免重复调用

### 8.3 数据安全
- 用户数据隔离（Row Level Security）
- 不存储任何对话的敏感信息
- 用户可一键删除所有个人数据
