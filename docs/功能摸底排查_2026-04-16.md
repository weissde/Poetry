## 2. 已实现的前端交互功能清单

## 2.1 公共与鉴权
- 登录/注册（邮箱密码），含会话初始化、错误提示、成功提示。
- 受保护路由拦截：未登录访问学习/练习/考试/记忆/我的学习/创作/图谱会跳转登录页。
- 会话过期处理：401 后自动清理缓存并引导重新登录，支持 `returnTo` 回跳。

## 2.2 首页与探索
- 首页：学习路径卡片、快速入口、推荐诗词入口跳转。
- 探索页：
- 关键词搜索、学段筛选（小学/初中/高中/全部）。
- 主题预设快捷搜索。
- 最近搜索（`localStorage` 持久化）。
- 结果视图切换（卡片/列表）。
- 高级筛选（朝代、题材多选）。
- 分页与每页条数切换。
- 单诗进入学习页、加入记忆队列。

## 2.3 学习台（Learn）
- 加载诗词详情、学习状态（收藏/笔记）、相关推荐。
- AI 流式解析（SSE）：支持简版/标准版/考试版深度。
- 交互式译解：原文与译文联动高亮。
- 语义拆解分栏：意象/情感/手法。
- TTS 朗读：当前句、全诗、可调语速、停止朗读。
- 学习动作：加入记忆、收藏切换、笔记保存。
- 相关推荐：可跳转到对应学习或练习。
- AI 导师深聊区：多轮对话、摘要保存/筛选/复用、语音输入、Markdown 导出。

## 2.4 练习场（Practice）
- 题组配置：主题、题量、难度、题型。
- URL 参数自动开练：支持来源标识、错题主观专项包参数。
- 题目生成、逐题作答、下一题、同点重练。
- 作答结果回传与维度统计。
- 练习完成后诊断视图（含雷达图、题型表现排序）。
- 复盘导出：JSON、Markdown、错题集（JSON+MD）。
- 图谱对比来源练习可写入会话总结日志。

## 2.5 考试中心（Exam）
- 四步开考流：选模板 → 配参数 → 预览 → 开考。
- 主观题比例配置与主观题数量覆盖。
- 创建考试、客观/主观混合作答。
- 倒计时与到时自动交卷。
- 手动交卷评分与成绩诊断。
- 复盘卡：一键复制、Markdown 导出。
- 历史中心：
- 历史记录分页、模式筛选、页大小切换。
- 单场弱项、与上一场对比、主观题趋势。
- 错题明细联动到错题本。

## 2.6 记忆打卡（Memory）
- 今日复习队列加载（到期/全部切换、分页）。
- 今日统计与总览指标加载。
- 复习工作区：
- 自评打卡（1-5 质量分）。
- 翻转卡片（正反面记忆）。
- 训练模式：上句接下句、逐句填空、全文默写、听写。
- 提示系统（分级提示）与质量分建议。
- 提交复习后自动刷新队列与统计。
- 加入记忆：诗词搜索后可加入。
- 打卡热力图（`react-calendar-heatmap`）。
- 成就与最近记录视图。

## 2.7 创作坊（Create）
- 三大工作区：创作点评、白话转诗、历史润色。
- 风格选择、参考诗词输入、自动高度文本框。
- 点评/转写结果展示：评分环、亮点/建议面板。
- 一键再润色（带润色指令）。
- 差异对比（Diff Viewer）。
- 历史记录：搜索、风格筛选、分页、回载到工作区。
- 分享闭环：`html2canvas` 生成海报并下载 PNG。

## 2.8 知识图谱（Graph）
- 加载多图谱数据：诗人图、意象图、个人薄弱图、时间轴、洞察。
- 工作区切换：总览/关联诗词/图谱详情。
- 诗人力导向图（`react-force-graph-2d`）：
- 节点检索定位。
- 节点点击聚焦。
- 放大/缩小/复位控制。
- 诗人关系边筛选（朝代）。
- 诗人对比：左右代表作与高频意象对比。
- 节点关联诗词：可跳学习、练习、错题本。
- 最近图谱对比日志：关键词、时间范围、排序、分页、删除、回到该对比。
- 时间轴与学习洞察模块（近 7 天行为指标）。

## 2.9 我的学习（MyLearning）
- 错题本工作区：
- 列表/统计/趋势三视图。
- 多维筛选：状态、题型、题目类型、关键词标签、朝代、题材、日期区间、关键词。
- 批量操作：批量改状态、批量删除。
- 单条操作：掌握/再练/删除。
- 收藏工作区：
- 收藏列表分页与关键词搜索。
- 取消收藏。
- 收藏笔记编辑与保存。
- 诊断工作区：
- 弱点画像读取。
- 最新考试诊断读取。
- 考试小结日志与图谱对比日志读取、筛选、删除。
- 计划工作区：
- 计划生成、最新计划加载。
- 任务勾选完成、同日重排、跨日移动。
- 浏览器通知提醒：权限申请、开关、测试提醒、定时提醒。

## 3. 功能规格（关键模块）

| 模块 | 触发/输入 | 核心输出 | 关键接口 |
|---|---|---|---|
| 登录注册 | 邮箱+密码提交 | 登录成功进入回跳页，失败显示错误 | Supabase Auth |
| 探索搜索 | 关键词/学段/分页/筛选 | 诗词列表、统计信息、筛选结果 | `GET /poems/search` |
| 加入记忆 | 探索/学习/记忆页点击“加入记忆” | 创建成功或已存在提示 | `POST /memory/enroll` |
| 学习解析 | 输入诗词+深度，点击解析 | 流式解析文本与结构化结果 | `POST /ai/analyze/stream` |
| 学习笔记 | 编辑笔记并保存 | 笔记落库、更新时间刷新 | `PATCH /poems/:id/note` |
| 收藏诗词 | 点击收藏/取消 | 收藏状态切换 | `POST /poems/:id/favorite` |
| 练习组题 | 配置主题/题量/难度/题型 | 题组生成并进入作答 | `POST /practice/questions/generate` |
| 练习作答上报 | 单题提交答案 | 统计更新，弱点画像可累积 | `POST /practice/answers` |
| 练习导出 | 点击 JSON/MD/错题导出 | 本地文件下载 | 前端本地生成 |
| 模考创建 | 选模板并开始 | 生成本场试卷 | `POST /exam/create` |
| 模考交卷 | 手动或倒计时到时 | 成绩、诊断、错题明细 | `POST /exam/submit` |
| 历史模考查询 | 页码/筛选变更 | 历史列表、趋势、对比 | `GET /exam/history` |
| 记忆队列 | 到期开关/分页 | 今日复习项列表 | `GET /memory/today` |
| 记忆打卡 | 质量分+模式提交 | 复习结果入库，队列刷新 | `POST /memory/review` |
| 创作点评 | 诗句+风格+参考诗 | 评分、亮点、建议、润色稿 | `POST /create/review` |
| 白话转诗 | 白话文本+风格 | 古风转写结果 | `POST /create/transform` |
| 再润色 | 指令+现有作品 | 新版本润色结果 | `POST /create/:id/refine` |
| 创作历史 | 关键词/风格/页码 | 历史版本列表 | `GET /create/history` |
| 图谱总览 | 打开图谱页 | 多源图谱数据加载 | `/graph/*` 系列接口 |
| 图谱对比日志 | 筛选/删除/回看 | 对比日志管理 | `GET/DELETE /practice/session-summaries` |
| 错题本批量 | 多选后批量操作 | 批量状态更新或删除 | `POST /wrongbook/batch` |
| 计划任务管理 | 勾选/排序/跨天移动 | 计划与进度同步更新 | `/review-plan/*` 系列接口 |

## 4. 前端 UI 设计盘点

## 4.1 视觉体系
- 主题基调：纸张感浅暖背景 + 墨蓝主色 + 金色强调。
- 设计令牌：CSS 变量统一管理颜色、阴影、圆角、控件高度、间距。
- 字体体系：`Noto Serif SC`（展示/诗词）+ `Noto Sans SC`（正文/控件）。

## 4.2 布局与组件模式
- 页面骨架：`Layout + Navbar + PageStage + SectionCard/SpotlightCard`。
- 交互组件：`PillNav`（分段切换）、`Magnet`（强调按钮）、`BlurText`、`TiltedCard`。
- 高信息页采用“工作区”分层：总览区 + 主任务区 + 详情区，降低认知负担。

## 4.3 动效与反馈
- 动效技术：Framer Motion + CSS 动画（入场渐显、骨架屏、状态过渡）。
- 反馈完整：加载中、空态、错误态、成功提示、禁用态、分页状态、复制成功状态。
- 可视化：雷达图、热力图、力导向图、趋势条图等多形态图表。

## 4.4 响应式与可用性
- 主要页面均有移动端到桌面端断点布局。
- 大部分关键操作提供替代路径（如复制失败可导出、无语音权限可文本输入）。

## 5. 技术框架盘点

## 5.1 前端
- 核心：React 18 + TypeScript + Vite 7。
- 路由：React Router v6（含受保护路由与懒加载）。
- 状态：Zustand（鉴权、练习等）+ 页面级 hooks。
- 样式：TailwindCSS + 全局 CSS 设计令牌。
- 动画：Framer Motion。
- 认证：Supabase Auth。
- 图形/可视化：`react-force-graph-2d`、`react-calendar-heatmap`、`recharts`。
- 导出与辅助：`html2canvas`、`lucide-react`。

## 5.2 前端数据层
- 统一 API 封装（`lib/api.ts`）：
- Token 缓存与自动注入。
- GET 缓存 + in-flight 去重。
- 超时、重试、错误规范化。
- 401 自动登出与登录回跳。
- SSE 流式请求支持（用于 AI 解析与聊天）。

## 5.3 后端（支撑层）
- 框架：FastAPI + Uvicorn。
- 数据/服务：Supabase、Redis。
- 核心能力：诗词检索、练习/考试、记忆、创作、图谱、错题与计划 API。
